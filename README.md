# AI Recommendation System - Project 3

## Description
This project is a simple AI recommendation system built using Python.  
It suggests items based on user interests using basic logic and tag matching.

## How it works
- User enters their interests (example: action, drama, tech)
- The system compares the input with item tags
- Items with matching tags are recommended to the user
- Results are sorted based on similarity score

## Files
- main.py → main program
- data.py → contains items and their tags
- recommender.py → recommendation logic

## How to run
1. Open terminal in project folder
2. Run: python main.py

## Example input
action, drama

## Output
Avengers -> Score: 1  
John Wick -> Score: 1  
Interstellar -> Score: 1  

## Skills used
- Python basics
- Logic building
- Lists and dictionaries
- Pattern matching
